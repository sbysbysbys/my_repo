　　原标题：《消失的她》拍摄地曝光！海水比三亚纯净椰林比普吉浪漫，趁没火快去！
　　作者：Feekr旅行
　　要说最近最火的电影，那当属《消失的她》！
　　总票房已经突破23亿，霸占豆瓣实时热门榜第一，连着上了多个微博热搜，成为了当之无愧的现象级电影。
　　而故事背景虽然发生在东南亚，但其实很多人都已经发现，这部电影大部分的镜头，都是在海南拍摄的……
　　而这一次，小飞君也挖到“她”的取景地之一 ——海南，万宁！
　　（其实电影在三亚陵水保亭也都有取景，不过万宁真的很值得安利！）
　　有人说，这里有更舒服的盛夏海浪+椰林日落；有人说，这是所有自由有趣灵魂的聚集地；还有人说，这里是个年轻的“野生乌托邦”。
　　七月悄然来临，盛夏翻开滚烫的诗篇，海南万宁，便在日落终点等你。
　　继续挖掘，才发现这里是众多电视剧、综艺的宠儿，《妻子的浪漫旅行》《十二道锋味》、《夏日冲浪店》、《完美的夏天2》、《去野吧，毛孩子》……都能在里面发现熟悉的场景。
　　万宁被美国时尚杂志评为“ 世界十大冲浪圣地”，尚未被开发的原生态海岛，能见度10米深的玻璃海，让这里成为冲浪和潜水爱好者的天堂。
　　高高的椰子树下，浪人们涂上彩色防晒泥 ，抱着五颜六色的冲浪板向碧海走去，冲浪服在阳光上泛着细碎光芒，凝聚成盛夏最美的回忆。
　　这里有无需修饰的富士蓝，长岛冰茶同色系的橙色落日，薄荷绿浪遮挡住夏日阳光，白沙滩上的粉色帐篷跳脱进眼球……用双眼，记录每一帧电影版的场景，记录多巴胺色系的海岛回忆。
　　“自由”，是万宁最吸引人的特质之一。
　　比起其他旅游城市，万宁有更多长期逗留的年轻人，开店、聊天、穿着沙滩裤、开着电单车吹海风，他们在充斥浓郁西海岸风格的沙滩上躺着晒太阳，在丛林里喝着不着边际的咖啡，这种看似“躺平”的状态，却是无数人的理想生活。
　　盛夏海浪、椰林日落
　　这些小众海岛，是时候C位破圈了！
　　/ 神州半岛 /
　　小编看《消失的她》的时候就觉得眼熟，这不是那个易梦玲同款的超火日落酒吧吗？！
　　电影中，男主带女主租潜水服的地方，正是万宁神州半岛超chill的科罗娜日落酒吧。这里可谓是万宁“地标”般的存在了，沙滩、海浪、日落、鸡尾酒…搭配出穿衣自由的辣妹天堂。下午6点左右，橙色日落降临，将沙滩晕染成浪漫的小加州，氛围感瞬间拉满。 
　　岛上，教堂、旋转木马、海上秋千、灯塔同样很好出片！带上墨镜、帽子、冲浪板，简直可以在这里拍出100种感觉。
　　/ 日月湾 /
　　日月湾，被美国时尚杂志评为“世界十大冲浪圣地”，是中国难得的国际性冲浪圣地，这也是为何越来越多浪人慕名前来的原因。
　　王一博的综艺《夏日冲浪店》便在此拍摄，今年言承旭新剧《夏花》也让这个小众宝藏地小火了一把。
　　这里冲浪氛围友好，每天出门就是放着冲浪板的白沙滩，以及各种五颜六色的冲浪店，冲浪将成为你日常生活的一部分。相比起三亚、海口等地，日月湾冲浪价格实惠，绝对是极具性价比的冲浪胜地。
　　万宁最美的海边教堂，就坐落于融创日月湾旁。在清晨朝霞笼罩下，太阳冲出海平线，海边教堂宛如披上一层柔和白纱，如诗如画。
　　/ 石梅湾 /
　　如果你喜欢自驾，请不要错过石梅湾旁35公里的沿海公路。一边椰林，一边海滩，纯净到极致的蓝与绿交织出最美公路。
　　如果你喜欢潜水，请不要错过这里如水晶般透亮的海水。当你被五彩珊瑚包围，宛若亲临海底花园。
　　如果你想体验当地人的生活，大胆去做吧，这里沙滩柔软，阳光温暖，尽情躺在沙滩上晒太阳谈天，体验无边无际的自由。
　　/ 加井岛 /
　　加井岛有着全海南最纯净的玻璃海，能见度10米深，是唯一媲美巴厘岛的所在，同时它也是一座原生态未被开发的无人岛，人们享受着最初的海岛时光。
　　在加井岛，你可以看到海底的礁石、珊瑚和鱼群，这里还有一艘玻璃船和网红之翼。
　　同样，这里也是世界级的潜水天堂，真正热爱潜水的人每年都会来到加井岛，放松，度假，潜水。
　　/ 大花角 /
　　尽管大花角只是一片野海滩，并不被认为是一个景点，但它却有着独特的魅力。当你驱车行至道路尽头，听着海浪的声音走向海边，你会看到一些渔船停泊在海面上。如果时间允许，还有机会碰到一些渔民正在打渔归来。
　　慢慢走近眼前是一整片鹅卵石滩，海水瓦蓝，自顾自拍打岸边的鹅卵石，把它们叠得整整齐齐，忽然一种强迫症的舒爽瞬间被治愈~
　　tips：导航“大花角”，车可以直接开到海边，停车的地方能看到渔村。这里被不少人吐槽过：来路不平顺，穿过林子过泥路，路上要避让鸡鸭。所以根据个人兴趣选择前往，注意安全，并在离开时，带走垃圾，保护好环境。
　　/ 山钦湾 /
　　山钦湾，一片小众而绝美的野海。这里遍布黑礁石和岩洞，巨石屹立，浪潮拍礁，海风呼啸，海水无人打扰地奔放着涌向沙滩。
　　幽然神秘的燕鸟洞，令人叫绝。站在山钦湾的巨石上发呆，看浪花拍岸，看阳光砸在海面上，叮叮当当，溅起一片灿烂。
　　小城不止海海海
　　解锁万宁的不冲浪的N种玩法
　　/ 桥海牧场 /
　　万宁的浪漫，从来不只有沙滩、大海和椰林。
　　来到桥海牧场，仿若走进宫崎骏的治愈系油画，饱满的绿色溢入眼中。
　　被大自然环抱的一大片湿地，一条条蜿蜒的木栈道，随处可见农户散养的小动物们，大白鹅们排着队钻过木栅栏，小鸭子在水坑里洗澡，简直就是向往的田园生活。
　　孤独的树在牧场里默默屹立，如同自由的灵魂，缓慢而坚定的生长。在桥海牧场，心情和身体都毫不费劲地轻松了起来。
　　/ 凤凰九里书店 /
　　来到万宁，你一定要去九里书店看一次书。
　　避开人声鼎沸，身在喧嚣却不孤独，与来自各地的读书者一起，放慢时针，细品文字。
　　择一片书签，都是独特的海水味。找到一个地方坐下阅读，瞬间整个世界都chill下来~
　　这里又被称为是“海南最美书店”，在书店里喝着咖啡看着书，抬头间就是白色的沙滩和蔚蓝的大海，心，不由自由静了下来。
　　咖啡续命，嗦粉上头
　　打卡万宁舌尖上的日日夜夜
　　/ 兴隆咖啡 /
　　如果你在万宁生活，看着日落，喝着兴隆咖啡，将会是你的日常。
　　兴隆镇，中国最好的咖啡产地之一。兴隆咖啡，则是海南省万宁市特产，中国国家地理标志产品。
　　《十二道锋味》便是在兴隆的隆苑咖啡屋取景，这里咖啡融入了人们的生活日常，随处可见的咖啡文化，是喜欢喝咖啡的人的天堂。
　　在兴隆，你可以在咖啡庄园体验自己研磨咖啡豆，学习咖啡冲泡知识，深入了解咖啡文化。
　　不同豆子由浅到深颜色的变化，代表着咖啡豆不同烘焙程度的成色。不同的豆子依据它的特性可能适合不同的烘焙程度，才能最大程度地呈现它的风味口感甚至尾韵余调。
　　园区内除了咖啡豆，还种植了木瓜、菠萝蜜、可可果等植物。园里手工彩绘的绝美指引牌，自然灵动又充满野趣，跟庄园的树林园地融合得恰到好处。
　　/ 海南粉 /
　　芳姐陵水酸粉，位于万宁石梅湾大道石梅湾。店面朴素，便是最常见的那种街头小店，每天却都排着长队。陵水酸粉味道酸辣甜香，用青柠提味，加上他们家特制的辣椒酱，酸粉王者诞生了！
　　后安汤粉也是来到万宁必打卡的美食，熬上一碗鲜美的猪头汤，以猪瘦肉、猪大肠、酥炸虾米、为配料，入口香脆，重复当一周的早餐也不过瘾！
　　 嗲嗲的椰子
　　这家椰子鸡算是众所周知的明星店了，林更新、姜文、宋妍霏、窦骁等都来打卡过！ 
　　和椰子清新的绿色一样，椰子鸡火锅的汤底鲜甜爽口，一口下肚，舌尖仍然留存着淡淡椰香。
　　推荐竹荪木瓜嗲嗲鸡、招牌凤尾虾和黑松露海南鸡饭，当然他们的绵绵冰也是人手一份。一定要多喝点汤，椰子的清甜和果香融为一体，如同海南蓬蓬勃勃的绿意，在唇齿间掀起一片海岛的浪潮。
　　想去万宁住一整个夏天，每天拎着冲浪板在日月湾晒到黝黑，闲来无事便去石梅湾自驾兜风，偶尔去加井岛潜水跟找上次遇到的珊瑚礁打声招呼。
　　海风洒脱，布满山河；风恬浪静，波澜不惊；人生海海，不过尔尔。
　　这个夏天，我在“野生乌托邦”万宁等你。
